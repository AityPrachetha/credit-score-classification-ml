# Best Performing Model
Random Forest Classifier achieved the highest accuracy of **78.37%**, making it the best model for Credit Score Classification.

# Conclusion
This project demonstrates how Machine Learning algorithms can be used to classify customer credit scores based on financial and banking details. Different models were trained and evaluated, and Random Forest provided the best performance due to its ensemble learning capability and ability to handle complex patterns in the dataset.
